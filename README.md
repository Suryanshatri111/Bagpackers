📄 Enquiry Page – Touring Website

This project includes an enquiry page for a touring website that allows users to easily submit their travel-related queries. The page is designed to improve customer interaction and streamline communication between users and the tour service provider.

✨ Features
🧑‍💻 Simple and user-friendly enquiry form
📱 Responsive design (mobile & desktop)
✅ Form validation for required fields
📩 Collects user details:
Name
Email
Phone number
Travel destination
Message / enquiry
🔔 Success and error notifications
🔗 Backend-ready (can be integrated with APIs or databases)
🎯 Purpose

The enquiry page helps users:

Ask about tour packages
Check availability and pricing
Request customized travel plans

It enhances user engagement and helps businesses respond quickly to customer needs.

🛠️ Technologies Used
HTML
CSS
JavaScript
(Optional: Backend like Node.js / PHP / Firebase)
